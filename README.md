### 🧑‍💻 About Me

東吳大學資料相關科系學生，專注於把**統計方法、機器學習與大型語言模型 (LLM)** 應用在真實問題上 —— 從金融市場的量化交易與情緒預測，到電商顧客行為、社群網路結構分析。
